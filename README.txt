PROYECTO: SISTEMA DE GESTION:
------------------------------------------


DESCRIPCION GENERAL 
------------------------------------------
El protecto es un sistema de gestion para una biblioteca universitaria
desarrollado en Python aplicando los principios de la POO y SOLID.
Permite administrar recursos bibliograficos, usuarios, prestamos,
devoluciones, multas y notificaciones.
Objetivo


FUNCIONALIDADES PRINCIPALES:
------------------------------------------
1. Registro de recursos de distintos tipos.
2. Registro de diferentes tipos de usuarios con parametros diferentes.
3.Creacion de prestamos segun disponibilidad de recursos y pagos de multas.
4. Devolucion de prestamos con calculo de multas si aplica.
5. Recordatorios a los usuarios de que debe entregar el recurso en el dia actual.
6. Escritura a archivos externos de texto JSON de los objetos.
7. Manejo de errores
